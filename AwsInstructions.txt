Instrucciones para despliegue en aws

1. crear una vpc(rayhov1) con dos subredes y un Nat gateway 

2. crear 3 security groups 
# 1. client-sg (sin cambios)
Inbound:
- HTTP (80) desde 0.0.0.0/0
- HTTPS (443) desde 0.0.0.0/0  
- SSH (22) desde tu IP
- TCP(5173) desde internet

# 2. Head-sg (comunicación con cliente)
Inbound:
- cualqueir comunicacion desde client-sg

# 3. workers-internal-sg (comunicación entre workers)
Inbound:
- cualquier comunicacion desde head-sg

3. crear la instancia head (usar ubuntu)
ponerlo en surbred privada y head-sg 
configurar el sript de inicio con baseconfig.bash

4. crear la instancia client 
ponerlo en subred publica y clinet-sg
configurara el script de inicio clientconfig.bash

--nota aqui ya se puede probar la ejecucion seq y paralela conectandose
via ssh a esta instancia y ejecutando el script peticion.py con la ip privada 
del la instancia head
